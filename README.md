🎮 Tic Tac Toe – Modern Web Game

A modern, interactive Tic Tac Toe web application built using HTML, CSS, and JavaScript with advanced UI, sound effects, and AI gameplay.

🚀 Features
🎯 Player vs Player Mode
🤖 Player vs Computer (AI Mode)
🔄 Restart Game Functionality
↶ Undo Move Feature
🔊 Sound Effects Toggle (ON/OFF)
✨ Winning Animation & Highlight Effects
🎨 Modern UI with Gradient Animations
📱 Fully Responsive Design (Mobile + Desktop)
🧠 Game Logic
Implements turn-based gameplay (X & O)
Detects:
✅ Win conditions (rows, columns, diagonals)
🤝 Draw conditions
AI Logic:
Tries to win if possible
Blocks opponent moves
Prioritizes center and corner positions
🛠️ Tech Stack
Frontend: HTML, CSS (Advanced Animations)
Logic: Vanilla JavaScript
Audio: Web Audio API
📂 Project Structure
📁 Tic-Tac-Toe
 ├── index.html   # Main game file (UI + Logic)
 ├── win-sound.mp3 (optional)

Clone the repository:

Click on any cell to make a move
Switch between:
Player vs Player
Vs Computer
